所谓并查集就是实现了合并和查找两个查找的集合。
我们常常用不同个数字来表达属于不同的集合，一般在数组中用他自己在数组中的位置来表示。
一开始所有的点都是他自己的组长，如果两个点具有“亲戚关系”（在图中就是两个点是联通的）那么我们就将两个点的集合合并
通常用如下函数来实现：
void merge（int x， int y，int *a）
{
   a[find(x)]=find(y);
}
其中find是一个函数，他的含义是找出x所属的集合，
int find（int x）
{
 if（x==a[x]）return x；
 return a[x]=find(a[x]);
 }
 注意最后一句话，他具有压缩路径的作用，可以大大加快速度
